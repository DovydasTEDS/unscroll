# unScroll - make doomscrolling hell

are you a victim of doomscrolling? did you say no? stop lying....  
this browser extension makes doomscrolling VERY hard... and frustrates you.  
  
here is a list of available *troll* options.  

- crazy volume - change you volume quickly and randomly
- tab closer - closes your doomscroll tabs every so often  
- upScroll - scroll up
- home - takes you to the site home page
  
here is a list of the supported websites:

- tiktok
- youtube shorts
- instagram reels
